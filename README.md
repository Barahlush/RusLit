# RusLit

![Chekhov and Tolstoy, 1901](Tolstoy_and_Chekhov_1901.jpg)

## Description

This repository contains a collection of Russian literature in txt format (UTF-8 encoding). 

This can be useful for data analysis - used as a dataset.
